# Formador de Oraciones (Estilo Duolingo con Pictogramas ARASAAC)

Esta aplicación es una plataforma educativa interactiva diseñada especialmente para niños, orientada al aprendizaje del idioma español y la estructuración correcta de oraciones utilizando la metodología de comunicación aumentativa.

## ¿De qué trata?
Inspirada en la dinámica de juego y progresión de **Duolingo**, la aplicación ayuda a los niños a formar oraciones arrastrando y ordenando elementos visuales. Utiliza los pictogramas oficiales de **ARASAAC** (Sistemas Aumentativos y Alternativos de Comunicación), lo que la hace sumamente accesible para la educación infantil, educación especial y apoyo al desarrollo del lenguaje.

## Características principales
- **Gamificación**: Lecciones y niveles progresivos para formar oraciones simples y complejas.
- **Pictogramas ARASAAC**: Apoyo visual premium para facilitar la asociación de conceptos y el aprendizaje intuitivo.
- **Accesibilidad**: Interfaz visual adaptada para niños y necesidades educativas especiales.
- **Formato Standalone**: Código optimizado y auto-contenido de rápida carga.
